This program includes a simple function add that takes two integers and returns their sum. In the main function, it declares two integers, calls the add function, and prints the result. Additionally, it uses a loop to print numbers from 1 to 5. 

        #include 

            int add(int a, int b) {
                return a + b;
            }
            
            int main() {
                int num1 = 5, num2 = 7;
                printf("Sum of %d and %d is: %d\n", num1, num2, add(num1, num2));
                printf("Numbers 1 to 5: ");
                for (int i = 1; i <= 5; ++i) {
                  printf("%d ", i);
             }
             printf("\n");

              return 0;
            }
